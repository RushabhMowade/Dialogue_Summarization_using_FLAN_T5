# 🗣️ Dialogue Summarization using FLAN-T5

## 📌 Overview
This project implements **dialogue summarization** using the **FLAN-T5 model from Hugging Face Transformers**.  
It explores **zero-shot,one-shot and few-shot inference**, applies **prompt engineering**, and evaluates model performance using **ROUGE metrics**.

---

## 🚀 Features
- Implemented **FLAN-T5 (Seq2Seq)** for dialogue summarization  
- Performed **zero-shot and few-shot inference**  
- Applied **prompt engineering** to improve summary quality  
- Evaluated results using **ROUGE-1, ROUGE-2, ROUGE-L**   

---

## 🛠️ Tech Stack
- Python  
- Hugging Face `transformers`  
- `datasets`  
- `evaluate` & `rouge_score`  
- PyTorch
